# vim-plang

This is a Vim syntax plugin for the [P][p_homepage] language.

## Installation

### Using [Vundle][v]

1. Add `Plugin 'dijkstracula/vim-plang'` to `~/.vimrc`
2. `vim +PluginInstall +qall`

### Using [Pathogen][p]

1. `cd ~/.vim/bundle`
2. `git clone https://github.com/dijkstracula/vim-plang.git`
